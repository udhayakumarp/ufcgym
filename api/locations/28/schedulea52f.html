Youth BJJ (No GI)
Class Description
Close Description
Students will learn that there is a time and a place for martial arts training. The program integrates; self-defense techniques, conditioning and skill oriented games aiming to improve your child’s motor skills, agility and flexibility. Youth BJJ is a fun class for youth that teaches discipline and helps build confidence. Youth BJJ No Gi classes are less technical and offer a faster and more dynamic style of fighting. Students do not qualify for belt or stripe ranks in the Youth BJJ No Gi program.
