Class Description
Close Description
UFC FIT class offers a total body workout that follows a set series of drills completed in six five-minute rounds accompanied by a warm-up and cool down. The drills are structured to challenge cardio capacity, build strength, and improve balance and flexibility. UFC FIT class offers the perfect full-body high-intensity interval training (HIIT). UFC FIT class is great for cross training and provides a unique mix of functional training and cardio that challenges the entire body in true MMA style.
