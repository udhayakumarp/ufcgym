The MATRX class is uniquely structured to combine TRX suspension bodyweight exercises and our UFC GYM MMA cardio conditioning in order to build endurance, strength, balance, flexibility and core stability simultaneously. A basic class includes cardio calisthenics, suspension training, shadow kickboxing, sprawls, technical standing, break falling, pummeling, sitting through, bag work (standing and ground), and core strengthening exercises. The 150 lb. heavy bags and TRX suspension trainer are the primary focus of the class.
